## 问题与处理

### 外部盘只读

外部盘原文件系统为 `vfat`。在其中克隆 Git 仓库并编译动态库时，由于 vfat 不支持
Linux 符号链接，目录结构损坏，内核将文件系统自动设置为只读。

处理过程：

1. 使用 `fsck.vfat` 确认并修复损坏。
2. 将外部盘重新格式化为 ext4。
3. 固定挂载到 `/mnt/zhufeng_data`。
4. 验证 ext4 外部盘可正常读写。

后续不得将 Linux 项目工作盘格式化为 vfat。

### Python 绑定无法导入

`pyrealsense2` 安装在 `/usr/local/lib/python3.6/pyrealsense2`，不在默认搜索路径。
通过以下 `.pth` 配置解决：

```bash
/usr/local/lib/python3.6/dist-packages/pyrealsense2.pth
```
