# Jetson 视觉项目部署说明

## 上传位置

将本目录中的全部内容上传到 Jetson 外部存储：

```text
/mnt/zhufeng_data/zhufeng/vision_jeason
```

上传后请确认项目目录位于外部 ext4 存储，而不是 eMMC：

```bash
cd /mnt/zhufeng_data/zhufeng/vision_jeason
findmnt -T .
df -h .
```

`findmnt` 输出中的设备应为 `/dev/sda1`，文件系统应为 `ext4`。

## 基础依赖

在 Jetson 上进入项目目录后安装依赖：

```bash
cd /mnt/zhufeng_data/zhufeng/vision_jeason
python3 -m pip install -r requirements-jetson.txt
```

如果 `pupil-apriltags` 在 Jetson Nano 上安装失败，先不要改代码，把完整报错贴出来；AprilTag 检测库需要根据 Python 3.6 和 aarch64 环境选择可编译版本。

## 相机验证

运行 30 帧 D435 对齐采集测试：

```bash
python3 tools/check_camera.py --frames 30
```

程序应打印彩色图尺寸、深度图尺寸和中心点深度。

## AprilTag 6D 位姿串口服务

本轮 ID 约定：

```text
tag_base_ref = ID 0
tag_tool0    = ID 1
```

当前使用的 AprilTag family 为 `tag25h9`，也就是生成器中的 `AprilTag Classic 25h9`。黑色外框边长为 `80 mm`。

返回值为 `tag_tool0` 在 `tag_base_ref` 坐标系下的 6D 位姿：

- `position_m`：位置，单位米。
- `orientation_xyzw`：姿态四元数，顺序为 x、y、z、w。
- 每次查询默认连续采集 15 帧，并对有效帧做稳健融合。
- 如果当前位姿看不到底座 ID0，但之前看见过底座 ID0，程序会使用最近多个底座参考缓存的稳健平均继续计算。
- JSON 中 `quality.base_ref_source` 会标明底座参考来源：`live`、`cached`、`mixed` 或 `none`。

串口协议：

```text
电脑端发送：@GET_TAG_POSE#
Jetson 返回：一行 sukinee_tag_pose_v1 JSON
```

不显示窗口的正式运行命令：

```bash
python3 tools/serve_tag1_pose_serial.py \
  --serial-port /dev/ttyTHS1 \
  --baudrate 115200 \
  --tag-size-mm 80 \
  --base-tag-id 0 \
  --tool-tag-id 1 \
  --sample-frames 15 \
  --min-valid-frames 5 \
  --base-cache-items 20
```

Jetson 有屏幕时，调试运行命令可以加 `--show`：

```bash
python3 tools/serve_tag1_pose_serial.py \
  --serial-port /dev/ttyTHS1 \
  --baudrate 115200 \
  --tag-size-mm 80 \
  --base-tag-id 0 \
  --tool-tag-id 1 \
  --sample-frames 15 \
  --min-valid-frames 5 \
  --base-cache-items 20 \
  --show
```

`--show` 会打开 `Sukinee AprilTag Debug` 窗口，显示：

- D435 彩色画面。
- 检测到的 AprilTag 外框。
- `ID 0 base_ref` 和 `ID 1 tool0` 标签文字。
- `base_ref_source`：`live`、`cached`、`mixed` 或 `none`。
- 最近一次串口查询状态。

在显示窗口中按 `q` 或 `Esc` 可以退出服务。

如果使用 USB-TTL 模块插在 Jetson 上，串口设备可能是 `/dev/ttyUSB0`；如果使用 Jetson 板载 3.3V TTL 串口，优先尝试 `/dev/ttyTHS1`。

返回示例：

```json
{"protocol":"sukinee_tag_pose_v1","sample_id":"S0001","seq":1,"timestamp_jetson":"2026-06-18T12:00:00.000","from_frame":"tag_base_ref","to_frame":"tag_tool0","tag_family":"tag25h9","tag_base_ref_id":0,"tag_tool0_id":1,"tag_size_m":0.08,"frame_count_used":15,"crc32":null,"position_m":[0.1,-0.2,0.3],"orientation_xyzw":[0.0,0.0,0.0,1.0],"quality":{"both_tags_seen":true,"base_ref_seen":false,"tool0_seen":true,"base_ref_source":"cached","mean_reprojection_error_px":null,"decision_margin_min":null,"hamming_max":null}}
```

如果末端 tag 不可见，或者既看不到底座 tag 又没有底座缓存，Jetson 仍会返回 JSON，但 `position_m` 和 `orientation_xyzw` 为 `null`，并在 `quality` 中标出失败原因。
