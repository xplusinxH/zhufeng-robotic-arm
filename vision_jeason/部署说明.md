# Jetson 视觉项目部署说明

## 上传位置

将本目录中的全部内容上传至 Jetson 外部存储：

```text
/mnt/zhufeng_data/zhufeng/vision_project
```

上传后目录应直接包含 `main.py`、`config.yaml`、`camera` 和 `tools`，不要额外嵌套
一层 `vision_jeason`。

## 确认存储位置

在 Jetson 中运行：

```bash
cd /mnt/zhufeng_data/zhufeng/vision_project
findmnt -T .
df -h .
```

输出中的设备应为 `/dev/sda1`，文件系统应为 `ext4`。如果设备显示
`/dev/mmcblk0p1`，说明文件位于 eMMC，应停止后续操作。

## 部署配置

项目文件上传完成后，将运行配置复制到 eMMC：

```bash
cp config.yaml /etc/zhufeng-vision/config/config.yaml
```

后续标定结果保存到：

```text
/etc/zhufeng-vision/calibration
```

## 相机验证

运行 30 帧对齐采集测试：

```bash
python3 tools/check_camera.py --frames 30
```

程序应持续打印彩色图尺寸、深度图尺寸和中心点深度。

## 保存相机内参

```bash
sudo mkdir -p /etc/zhufeng-vision/calibration
sudo chown jetson:jetson /etc/zhufeng-vision/calibration
python3 calibration/calibrate_intrinsic.py
```

内参将保存到：

```text
/etc/zhufeng-vision/calibration/camera_intrinsic.json
```

## 实时 RGB-Depth 对齐验证

在 Jetson 屏幕中运行：

```bash
python3 tools/check_alignment.py
```

操作方式：

- 鼠标左键点击左侧彩色图：显示深度和相机三维坐标。
- `S`：保存当前验证数据。
- `Q` 或 `Esc`：退出。

验证数据保存到：

```text
/mnt/zhufeng_data/zhufeng/data/alignment
```
