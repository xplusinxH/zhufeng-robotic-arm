# Jetson 视觉项目部署说明

## 上传位置

将本目录中的全部内容上传到 Jetson 外部存储：

```text
/mnt/zhufeng_data/zhufeng/vision_jeason
```

上传后请确认项目目录位于外部 ext4 存储，而不是 eMMC：

```bash
cd /mnt/zhufeng_data/zhufeng/vision_jeason
findmnt -T .
df -h .
```

`findmnt` 输出中的设备应为 `/dev/sda1`，文件系统应为 `ext4`。

## 基础依赖

在 Jetson 上进入项目目录后安装依赖：

```bash
cd /mnt/zhufeng_data/zhufeng/vision_jeason
python3 -m pip install -r requirements-jetson.txt
```

如果 `pupil-apriltags` 在 Jetson Nano 上安装失败，先不要改代码，把完整报错贴出来；AprilTag 检测库需要根据 Python 3.6 和 aarch64 环境选择可编译版本。

## 运行配置

配置文件可以保存在 eMMC：

```bash
sudo mkdir -p /etc/zhufeng-vision/config
sudo cp config.yaml /etc/zhufeng-vision/config/config.yaml
```

标定结果保存到：

```text
/etc/zhufeng-vision/calibration
```

## 相机验证

运行 30 帧 D435 对齐采集测试：

```bash
python3 tools/check_camera.py --frames 30
```

程序应打印彩色图尺寸、深度图尺寸和中心点深度。

## 保存相机内参

```bash
sudo mkdir -p /etc/zhufeng-vision/calibration
sudo chown jetson:jetson /etc/zhufeng-vision/calibration
python3 calibration/calibrate_intrinsic.py
```

内参会保存到：

```text
/etc/zhufeng-vision/calibration/camera_intrinsic.json
```

## RGB-Depth 对齐验证

在 Jetson 屏幕中运行：

```bash
python3 tools/check_alignment.py
```

操作方式：

- 鼠标左键点击左侧彩色图：显示深度和相机三维坐标。
- `S`：保存当前验证数据。
- `Q` 或 `Esc`：退出。

验证数据保存到：

```text
/mnt/zhufeng_data/zhufeng/data/alignment
```

## AprilTag 末端坐标串口服务

临时任务使用：

- `tag0`：固定在机械臂底座或底座附近。
- `tag1`：固定在执行器末端。
- 返回值：`tag1` 原点在 `tag0` 坐标系下的位置，单位毫米。

串口协议：

```text
查询：@GET_TOOL#
有效返回：@TOOL,x_mm,y_mm,z_mm,age_ms#
无有效识别：@NO_TAG#
错误：@ERR,message#
```

运行示例，假设 AprilTag 黑白图案边长为 60 mm：

```bash
python3 tools/serve_tag1_pose_serial.py \
  --serial-port /dev/ttyUSB0 \
  --baudrate 115200 \
  --tag-size-mm 60
```

如果 tag0 和 tag1 的实际打印边长不同，先不要混用同一个 `--tag-size-mm`，需要告诉我两个实际尺寸，我会把程序改成分别配置 `tag0` 和 `tag1` 的尺寸。
